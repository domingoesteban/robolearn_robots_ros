## ROS launch files
Visualization in Rviz:
    roslaunch **[robot_name]**_description **[robot_name]**_rviz.launch
    
Spawn and visualize in Gazebo:
    roslaunch **[robot_name]**_gazebo **[robot_name]**_world.launch
